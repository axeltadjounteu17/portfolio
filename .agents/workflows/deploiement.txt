🚀 Comment déployer sur Vercel (le plus rapide)
bash# 1. Build ton projet
npm run build

# 2. Push sur GitHub
git init
git add .
git commit -m "portfolio v1"
git push origin main
```
Ensuite sur **vercel.com** → Import Git Repository → sélectionne ton repo → **Deploy**. C'est tout.
```
